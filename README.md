・ubuntu-20.04.3<br>
・ROS

# 使い方
以下を入力してインストール
'''
cd ~/catkin_ws/src
git clone 
cd ~/catkin_ws
catkin_make
'''

roscpreを起動させる(&をつけないと起動させながらできない)
'''
roscore &
